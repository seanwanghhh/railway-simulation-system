# Railway Simulation System

A Java-based railway transportation simulation system supporting train operations, station management, route validation, and transportation scheduling.

## Features

- Train movement and routing simulation
- Station and railway track management
- Route validation and exception handling
- Object-oriented architecture using encapsulation and polymorphism
- Modular system structure for improved maintainability

## Technologies

- Java
- Gradle
- Object-Oriented Design

## Project Structure

```text
app/src/main/java
├── trains
│   ├── Train.java
│   ├── Station.java
│   ├── Track.java
│   └── TrainsController.java
├── response
├── utils
└── App.java
```

## Getting Started

Clone the repository:

```bash
git clone https://github.com/seanwanghhh/railway-simulation-system.git
```

Build the project:

```bash
./gradlew build
```

Run tests:

```bash
./gradlew test
```
